Missing Cluster Configuration


In order to configure kafka-metrics-ui you need to add env.js file in the root directory of the app.
Example env.js structure: 
        
            var clusters =
            [
            {
            NAME:"prod",
            KAFKA_LENSES_URL : "prod.url.com",// "The Kafka Rest url"
            },
            {
            NAME:"dev",
            KAFKA_LENSES_URL : "dev.url.com", "The Kafka Rest url"
            }
            ]
